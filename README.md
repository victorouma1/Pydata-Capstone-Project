# Pydata-Capstone-Project

### Problem Statement
Kenya faces increasing environmental challenges such as:<br>

Air pollution in urban areas like Nairobi<br>
Climate variability (irregular rainfall, droughts)<br>
Limited access to centralised environmental insights <br>

Despite the availability of data, it is often:<br>
<br>
Fragmented<br>
Hard to interpret<br>
Not visualized geographically<br>

This project solves that by integrating datasets into a single visual analytics platform.<br>

### Description
This project aims to build an interactive data-driven dashboard that monitors environmental conditions across Kenya—focusing on:<br>

Air quality<br>
Rainfall patterns<br>
Regional environmental disparities<br>

This dashboard is primarily designed for policymakers and environmental organisations

### Dataset Used
sensors.AFRICA Air Quality Archive Nairobi<br>
Kenya Rainfall Dataset (WRI)<br>
